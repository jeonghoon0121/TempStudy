📝 1. 기획 및 디자인
    • 본인이 기획한 웹사이트를 Figma를 사용하여 화면을 설계한다.
ㅇ
💻 2. 구현
    • Figma에 생성된 화면 기반으로 HTML, CSS, JAVASCRIPT를 활용한 웹 페이지를 제작한다.
    1) 시맨틱 태그를 사용하여 문서 구조를 명확하게 정의하고, 일반 HTML 태그도 적절히 활용한다.ㅇㅇ
    2) CSS를 사용하여 웹 페이지의 디자인과 레이아웃을 세부적으로 구성한다. ㅇㅇ 
        이 때, 스타일과 레이아웃이 Figma에서 설계한 디자인과 일치하도록 한다. ㅇㅇ 
    3) 사용자가 다른 페이지로 쉽게 이동할 수 있도록 링크를 포함한 메인 페이지를 제작한다. ㅇㅇ
    4) 메인 페이지 외 최소 2개의 상세페이지를 제작한다.
    5) 이벤트 핸들러를 이용하여 사용자가 특정 영역에 상호작용할 때 시각적 효과를 적용한다.
     6) 최소 2개의 이미지를 삽입하고, 이미지 최적화와 ‘alt’ 속성 설정을 통해 접근성을 고려한다.
