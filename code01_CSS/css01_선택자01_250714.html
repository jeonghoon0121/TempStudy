CSS 선택자

기본 선택자

특정 html 태그를 선택할 때 사용하는 기능으로 태그를 선택하여 원하는 스타일과 기능을 적용

모든 선택자

html 문서안의 모든 태그를 선택
*{}

태그 선택자

html 문서안의 같은 태그를 모두 선택
p{}

아이디 선택자

html 문서안의 해당하는 아이디의 속성 값을 가진 태그만 선택

  • idx classx
  • idx classx
  • id1 classx
  • id2 classx
  • id1 class1
  • id2 class2

아이디1 yellow 아이디2 green

클래스 선택자

html 문서안의 여러개의 태그를 동일한 클래스명으로 지정하여 선택

  • idx classx
  • idx classx
  • idx class1
  • idx class2
  • id1 class1
  • id2 class2

클래스1 aquamarine 클래스2 pink
