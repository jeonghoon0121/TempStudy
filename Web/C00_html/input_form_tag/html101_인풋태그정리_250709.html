인풋태그정리

text input tag

size= 화면에 몇글자가 보이도록 할것인지 지정 최대 10개 입력가능한 필등[ size가 5이면 5개만 보이고 나머지 5개는 보이지 않음

value= 텍스트필드에 보여지는 내용, 빈 텍스트필드에는 비어있음 비밀번호에서는 쓰이지 않음

maxlength= 텍스트 필드와 비밀번호 필드에서 입력가능한 최대 문자수

checkbox, radio, number, range

checkbox= 2개이상 토글기능/ radio=1개만 토글없음

value=선택된 체크박스,라디오버튼을 서버에게 알려줄때 값, 영문또는 숫자

checked=기본으로 선택하고싶을때

name= 라디오버튼과 체크박스는 하나의 그룹이므로 같은속성으로 지정


numbe : 숫자를 입력하는 스핀박스, 상품 개수등을 쓸때

range : 범위를 입력하는 슬라이드 막대, 볼륨조절등 정확할 필요는 없을때 값을 크거나 작게 변경할때, 바뀌는 값을 표시하려면 자바스크립트를 이용해야함

min : 필드에 입력가능한 최소값, 기본값 0

max : 필드에 입력가능한 최대값, 기본값 100

step : 숫자간격, 기본값 1

value : 필드에 표시할 초기값

날짜, 달력을 넣으려면



min : 시작 날짜, 범위제한

max : 마지막 날짜, 범위제한

step : 스핀 박스의 화살표를 클릭할경우 증감시킬 크기

value : 기본 표시 날짜
